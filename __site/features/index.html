Features

Below is a list of features of Pyblish, followed by features in planned for release sometime in upcoming versions.

Have your say about upcoming features, or suggest more, over at the GitHub repository

End-user Features

  • Software Agnostic
  • Standalone Support
  • Partial Publishing
  • Bi-directional Publishing
  • Detailed Error Summary
  • Quick Publish
  • Per Process Plugins
  • Per User/Organisation Plugins
  • Per Environment Plugins

Planned End-User Features:

  • Background Publishing
  • Command-line Support
  • Distributed Publishing
  • Concurrent/Multiprocessor Publishing
  • Scheduled Validation
  • Optional Plugins
  • Override Plugins
  • Nested Plugins
  • Repair
  • WIP Validation
  • Comments

Current Developer Features:

  • Killer Documentation
  • Standard PEP08
  • Native Python 2.6 - 2.7
  • LGPLv3
  • Extensible
  • Plug-in based
  • Hot Swapping
